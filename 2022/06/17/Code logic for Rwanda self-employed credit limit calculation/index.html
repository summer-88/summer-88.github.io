3.Calculating the credit limit

3.1 Determining whether a customer is an overdue user of lipa later

​ Check the MusoniLoan loan status by id_number and phone_number. if loan_status=Closed (written off) or in_arrears=true, then end the process with the response message.

{
 "limit_status": "limit_rejected",
 "limit_amount": 0.0,
 "credit_option": "rejected",
 "approval_type": "auto_rejection",
 "approval_date": null,
 "limit_description": "Customer has lipalater loans in arrears",
 "has_lipalater_arrears": true
}

3.2 Determine if the business payment type of self-employed is accepted

​ Determine if the field business_payment_type of the request is not in ACCEPTED_PAYMENT_TYPES[bank mobile_money], then end the process with the response message.

{
 "limit_status": "limit_rejected",
 "limit_amount": 0.0,
 "credit_option": "rejected",
 "approval_type": "auto_rejection",
 "approval_date": null,
 "limit_description": "Self employed customer payment type is unaccepted",
 "has_lipalater_arrears": false
}

3.3 Determining whether the total monthly income of a self-employed business exceeds its total expenses

​ Determine if the field total_expenses of the request exceeds the field business_gross_monthly_revenue of the request, then end the process with the response message.

{
 "limit_status": "limit_rejected",
 "limit_amount": 0.0,
 "credit_option": "rejected",
 "approval_type": "auto_rejection",
 "approval_date": null,
 "limit_description": "Self employed customer has no available disposable income",
 "has_lipalater_arrears": false
}

3.4 Rwanda Self employed customer required to provide bank statements

​ End the process with the response message.

{
 "limit_status": "limit_extra_docs",
 "limit_amount": 0.0,
 "credit_option": "pending_approval",
 "approval_type": "",
 "approval_date": null,
 "limit_description": "Rwanda Self employed customer required to provide bank statements",
 "has_lipalater_arrears": false
}
